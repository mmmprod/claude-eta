# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |
| < 0.1.0 | No        |

The `main` branch is also supported on a best-effort basis until the next tagged release.

## Reporting a Vulnerability

Do not open public GitHub issues for suspected vulnerabilities.

Use GitHub's private vulnerability reporting in the repository Security tab:

https://github.com/mmmprod/claude-eta/security

Please include:

- a clear description of the issue and its impact
- affected versions, tags, or commit SHAs if known
- reproduction steps or a proof of concept
- any suggested remediation, if available

Best-effort response targets:

- initial acknowledgement within 3 business days
- a follow-up status update within 7 business days after reproduction

If the report is confirmed, fixes will be prepared privately when appropriate and disclosed after a patch or mitigation is available.
